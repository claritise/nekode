# nekode

A tamagotchi-style desktop companion that shows you what Claude Code is doing. Built with an ESP32-S3, a tiny OLED display, RGB LEDs, and a microphone.

Even with a second monitor showing Claude Code, it's easy to miss when it needs input or finishes a task. nekode sits on your desk and gives you an at-a-glance status ‚Äî a cute animated kitty that reacts to Claude's state in real-time.

## How it works

[Claude Code hooks](https://code.claude.com/docs/en/hooks) send serial commands to the ESP32 whenever the agent's state changes. The kitty animates accordingly and the RGB LED strip reacts ‚Äî rainbow while working, solid white when waiting for input, off when idle.

```
Claude Code hook ‚Üí set_status.sh ‚Üí USB serial ‚Üí ESP32 ‚Üí OLED + LEDs
```

### State mapping

| State | Hook Event | Matcher | LEDs |
|-------|-----------|---------|------|
| typing | `UserPromptSubmit`, `PostToolUse` | ‚Äî | rainbow |
| reading | `PreToolUse` | `Read` | rainbow |
| typing | `PreToolUse` | `Edit\|Write\|NotebookEdit` | rainbow |
| running | `PreToolUse` | `Bash` | rainbow |
| searching | `PreToolUse` | `Grep\|Glob` | rainbow |
| browsing | `PreToolUse` | `WebSearch\|WebFetch` | rainbow |
| planning | `PreToolUse` | `TodoWrite` | rainbow |
| thinking | `PermissionRequest`, `PreToolUse` | `AskUserQuestion` | solid |
| error | `PostToolUseFailure` | ‚Äî | rainbow |
| done | `Stop` | ‚Äî | rainbow |
| spawning | `SubagentStart` | ‚Äî | rainbow |
| herding | `SubagentStop` | ‚Äî | rainbow |
| compacting | `PreCompact` | ‚Äî | rainbow |
| boot | `SessionStart` | `startup` | rainbow |
| sleep | `SessionEnd`, `Notification` | `idle_prompt` | off |
| waiting | after done finishes | ‚Äî | off |

Hooks are configured in `~/.claude/settings.json`. See the [hooks guide](https://code.claude.com/docs/en/hooks) for setup.

## Hardware

| Part | Details |
|------|---------|
| MCU | ESP32-S3-DevKitC-1 |
| Display | SSD1306 128x64 OLED (I2C) |
| LEDs | 6x APA106 F8 8mm RGB |
| Mic | INMP441 I2S microphone |

## Serial commands

Send newline-terminated commands over USB serial at 115200 baud:

```
thinking, typing, running, reading, searching, planning,
browsing, spawning, waiting, done, error, sleep, boot,
herding, compacting
```

## Build

Requires [PlatformIO](https://platformio.org/) and Python 3 with Pillow.

```
./build.sh           # generate headers from GIFs + build
./build.sh upload    # generate headers + build + flash
pio device monitor   # serial monitor
```

## Project structure

```
src/
  config.h         ‚Äî pin definitions, timing constants
  main.cpp         ‚Äî setup/loop, serial command dispatcher
  oled.h/.cpp      ‚Äî OLED display: neko animation + spectrum analyser
  led_strip.h/.cpp ‚Äî RGB LED strip: rainbow animation via RMT
  mic.h/.cpp       ‚Äî INMP441 microphone: I2S RMS volume
include/
  *.h              ‚Äî animation frames (generated by gif_to_header.py)
gif_to_header.py   ‚Äî converts GIF ‚Üí C header with PROGMEM frame arrays
set_status.sh      ‚Äî sends serial commands to ESP32
build.sh           ‚Äî one-step build: gif gen + compile + optional flash
```
